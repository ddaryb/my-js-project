## Сравнение производительности

1. Базовый вариант: 15 сек
2. Кэширование: 24 сек	
3. Кэширование + SonarQube: 1 мин 14 сек	
4. Проверка безопасности: 52 сек

## Выводы

1. Кэширование и SonarQube дают значительный прирост времени, на малых проектах их эффективность низкая.
2. Проверка безопасности замедляет процесс в 3,5 раза.
3. Для максимальной скорости лучше использовать базовую конфигурацию, так как дополнительные инструменты не дают преимуществ в этом случае.
